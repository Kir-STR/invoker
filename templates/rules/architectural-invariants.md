# Architectural invariants — инварианты проекта

Операционная выжимка по архитектурным границам проекта: что где лежит и что куда импортировать нельзя. Источник правды — спека / roadmap проекта. Описывает **целевое** состояние; реальное читается из исходного кода. Заполняется под проект; при переименовании / добавлении модулей — обновлять и здесь.

> **Как заполнять.** В каждом разделе замени `TODO: заполнить под проект.` на фактические границы своего кода по подсказке _курсивом_. Неприменимый раздел удали. Содержание выводится из реальной структуры `src`/модулей; при изменении структуры — обновляй здесь же.
